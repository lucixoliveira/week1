6 user stories for recyclers costs managing app

- As a user, I want to define the type of the payment I made
- As a user, I want to identify the vendor to whom I made the payment
- As a user, I want to insert the date in which I made a payment
- As a user, I want to insert the value of the payment I made
- As a user, I want to insert a brief description of the payment I made, so that I can keep track of my costs easily
- As a user, I want to the list of payments I made to be displayed after I saved them
