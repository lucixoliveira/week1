6 user stories for recyclers’ costs of managing app

- As a user, I want to insert the costs I have to operate my recycling company
- As a user, I want to insert the date in which I made a payment
- As a user, I want to insert a description of the payment I made, so that I can keep track of my costs easily
- As a user, I want to insert the value of the payment I made
- As a user, I want to signal if the item was a donation and therefore did not represent any cost
- As a user, I want to the list of payments I made to be displayed after I saved them
